ServMon@LUG

每分钟，我们会检测URL的状态，自动短信通知服务器故障。
为防止不稳定的服务导致短信泛滥，每个URL每24小时至多发送10条短信。

目前只支持HTTP协议

指定网页中的一个特征字符串，若未找到此字符串（如发生数据库错误）时将报警。

留空表示不检查。精确匹配。UTF-8 编码，其他编码网页请使用非中文部分作为特征。

接收报警短信使用，每个手机号至多监控10个URL。

    switch($status) {
        case 0: return '恢复正常';
        case 1: return '未检测到特征字符串';
        case 2: return '页面为空';
        case 3: return '连接超时';
        case 4: return 'HTTP状态码为'.$status_code;
        case 5: return '无法连接';
        case 6: return '服务器ping不通';
        case 7: return 'DNS解析失败';
        default: return '未知内部错误';
    }
